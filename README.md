# Keane Site Services - Premium Marketing Site

A high-end, single-page marketing website for Keane Site Services, built with React, TypeScript, and Tailwind CSS.

## Features

- **Premium Design**: Apple-level polish with industrial construction aesthetics
- **Hero Video**: Full-viewport looping background video with accessibility controls
- **Responsive**: Mobile-first design that works perfectly on all devices
- **Fast Performance**: Optimized for Core Web Vitals
- **SEO Ready**: Complete meta tags, structured data, and sitemap
- **Accessible**: WCAG AA compliant with proper focus management
- **Netlify Forms**: Working contact form integration
